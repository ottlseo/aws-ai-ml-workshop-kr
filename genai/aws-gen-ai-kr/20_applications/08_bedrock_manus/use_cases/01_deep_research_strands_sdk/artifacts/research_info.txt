### Topic 11: 코딩 환경 성능 벤치마크
- 프로세서 성능 비교 [20]
  * MacBook Pro M3: 코드 컴파일 및 빌드 속도 최상위
  * LG gram Pro: 지속적 CPU 부하 처리 우수
  * ASUS Zenbook: AI 가속으로 개발 생산성 향상
- 실제 개발 환경 성능 [21]
  * Visual Studio Code 실행 성능
    - MacBook Pro: 네이티브 최적화로 최고 성능
    - LG gram Pro: 안정적인 중상위권 성능
    - ASUS Zenbook: Intel CPU 최적화로 우수한 성능
  * IntelliJ IDEA 성능
    - 모든 모델에서 32GB RAM 구성 시 원활한 실행

### Topic 12: IDE 호환성 및 성능
- 주요 IDE 호환성 [22]
  * Visual Studio Code
    - 모든 플랫폼에서 안정적 실행
    - MacBook의 경우 네이티브 M3 최적화 버전 제공
  * Eclipse
    - Windows 기반 모델에서 더 안정적
    - MacBook의 경우 Rosetta 2 통해 실행
  * IntelliJ IDEA
    - 모든 플랫폼 네이티브 지원
    - RAM 사용량 높음 (16GB 이상 권장)

### Topic 13: 멀티태스킹 성능
- RAM 관리 및 성능 [23]
  * MacBook Pro M3
    - 통합 메모리 구조로 효율적인 메모리 관리
    - 16GB로도 32GB 수준의 성능 발휘
  * LG gram Pro
    - DDR5 메모리로 빠른 데이터 처리
    - 최대 32GB 지원으로 대규모 프로젝트 가능
  * ASUS Zenbook
    - LPDDR5X-7467 메모리로 고속 데이터 처리
    - 메모리 솔더링으로 업그레이드 제한

### Topic 14: 배터리 성능 테스트
- 개발 환경에서의 배터리 지속시간 [24]
  * MacBook Pro M3
    - IDE 실행 시 약 12시간
    - 컴파일 작업 시 약 8시간
  * LG gram Pro
    - IDE 실행 시 약 10시간
    - 컴파일 작업 시 약 6시간
  * ASUS Zenbook
    - IDE 실행 시 약 9시간
    - 컴파일 작업 시 약 5시간

### Topic 15: 키보드 및 디스플레이
- 키보드 품질 [25]
  * MacBook Pro
    - 1.0mm 키 트래블
    - 안정적인 타이핑감
    - 백라이트 균일도 우수
  * LG gram Pro
    - 1.5mm 키 트래블
    - 넘버패드 제공
